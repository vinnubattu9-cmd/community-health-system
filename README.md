# Community Health Monitoring & Awareness System

A full-stack web application to monitor community health, record vitals, calculate BMI, and generate health reports.

## Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB

## Features
- Health survey submission with live BMI calculator
- Downloadable PDF health reports
- Admin dashboard with Chart.js analytics
- Excel & PDF export of all records
- JWT-based admin authentication
